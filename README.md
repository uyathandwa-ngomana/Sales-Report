# Sales Dashboard - South Africa 2024

I built this dashboard to develop my Power BI skills using a 
real-world sales dataset. The data covers 50 transactions across 
4 South African provinces and 8 products from January to June 2024.

## What I did
- Imported and cleaned the data in Power BI Desktop
- Created a calculated Revenue column using DAX
- Built interactive visuals to analyse sales performance
- Added slicers so users can filter by region and category
- Published the final report to Power BI Service

## Key insights from the data
- Total revenue of R673K over 6 months
- Electronics drove 88% of total revenue
- Gauteng was the top performing region at R242K
- June was the strongest month at R190K

## Tools
- Power BI Desktop
- DAX
